# 期货分析系统 v5.0

> 全维度多因子期货分析系统，支持技术面 + 基本面 + 情绪面三维共振评分

## 🆕 v5.0 新增功能（第二阶段）

### 1. 期现价差分析 (`basis_analysis.py`)
- 期货价格 vs 现货价格实时对比
- 基差类型判断（正基差/负基差）
- 套利机会识别（强/中/弱套利机会）
- 市场结构分析（整体期现关系）

### 2. 机构持仓排名 (`institutional_positions.py`)
- 从交易所数据爬取主力持仓排名
- 识别主力加仓/减仓信号
- 跟踪大资金动向（净多/净空变化）
- 整体市场情绪判断（偏多/偏空/中性）

### 3. 信号强度评分系统 (`signal_scoring.py`)
- **多维度共振评分（1-10分）**
  - 技术面评分（40%）：RSI + MACD + 均线 + 布林带 + 量比
  - 基本面评分（35%）：库存趋势 + 期现价差 + 机构持仓 + 政策
  - 情绪面评分（25%）：VIX + 南华指数 + 美元指数 + 新闻情绪
- 三维共振检测（三维共振⭐⭐⭐ / 双维共振⭐⭐）
- 历史胜率统计（自动记录并验证信号准确率）

### 4. Kimi分析 v4.0 (`llm_analyzer.py`)
- 融入期现价差维度
- 融入机构持仓维度
- 融入信号评分维度
- 输出推荐等级：**强烈推荐 / 中性偏多 / 中性 / 中性偏空 / 谨慎**

## 🔄 数据采集流程（v5.0）

```
Step 1: 行情数据（双接口降级）
Step 2: 技术指标计算（RSI/MACD/布林带/均线）
Step 3: 库存原始数据
Step 3b: 库存环比趋势分析 ← 第一阶段
Step 3c: 经济数据日历 ← 第一阶段
Step 4: 政策宏观数据
Step 5: 市场情绪（VIX/南华/美元）
Step 6: 财经新闻
Step 6b: 地缘政治风险识别 ← 第一阶段
Step 6c: 库存-价格背离分析 ← 第一阶段
Step 7: 期现价差分析 ← 第二阶段 NEW
Step 8: 机构持仓排名 ← 第二阶段 NEW
Step 9: 信号强度评分 ← 第二阶段 NEW
```

## ⚙️ 配置说明

```python

KIMI_API_KEY = "你的Kimi API Key"
KIMI_BASE_URL = "https://api.moonshot.cn/v1"
MODEL_NAME = "moonshot-v1-8k"

# 邮件推送配置
EMAIL_HOST = "smtp.qq.com"
EMAIL_USER = "你的QQ邮箱"
EMAIL_PASS = "邮箱授权码"
EMAIL_TO = ["接收邮箱"]
```
